Data

Here (include link) you can download our entire dataset in one CSV file. This is a rather large file (879 mb) and contains over 2.7 million rows. The concatenated dataset combines the datasets of all 28 EU member state countries. We have collected all available ERDF, ESF and CF datasets for the 2007-2013 and 2014-2020 period. However, major gaps still exist for the 2014-2020 period, because not all data has been released.

The original files are described here For creating the joined datasets out of these files, we mapped every single dataset to common denominator variables that can be compared between the countries. More information on how we did this can be found under the Resources tab.
