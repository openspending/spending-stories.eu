# subsidystories.eu

## Quick start

- get the code
`git clone https://github.com/openspending/subsidystories.eu.git`

- install dependencies
`npm install`

- build the frontend assets
`npm run build`

- run the tests
`npm test`

- run the application...
`npm start`

- ... or just open index.html in your favorite browser

- check code style
`npm run review`
